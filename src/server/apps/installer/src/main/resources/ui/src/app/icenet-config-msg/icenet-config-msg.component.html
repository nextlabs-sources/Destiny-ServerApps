1. On the Management Server, copy the cacerts.jks file available at:
    .../Nextlabs/PolicyServer/server/certificates
  2. On the ICENet server, paste the cacerts.jks file in the following directory:
    .../Nextlabs/PolicyServer/server/certificates
  3. On the Management Server, copy the bootstrap.properties file available at:
    .../Nextlabs/PolicyServer/server/configuration
  4. On the ICENet server, paste the bootstrap.properties file in the following directory:
    .../Nextlabs/PolicyServer/server/configuration
